Claude に任せる

企業のURLリストを渡すだけで、フォームへの入力・送信・記録まで自動でやってくれます。このページの手順を上から順に進めるだけでOKです。

⏱ 所要時間の目安：30〜60分（初回のみ）

何ができるようになるの？

「Browser Use（ブラウザ・ユーズ）」というツールを使うと、Claude Code がパソコンのブラウザ（Chrome）を代わりに操作してくれます。 あなたは企業のURLリストを用意して指示を出すだけ。あとは Claude が自動でフォームを探して、内容を入力して、送信してくれます。

まず「黒い画面」を開こう

これから行う作業のほとんどは、「ターミナル」（黒または白い画面）という場所で進めます。 難しそうに見えますが、やることはシンプルです。
「決まった文字列を貼り付けて、Enterキーを押す」——これだけです。

💡
このページに出てくる黒い背景のコードは、全部コピーしてターミナルに貼り付ければ動きます。自分で打ち込む必要はありません。

📍 Macでターミナルを開く方法

  1. キーボードの ⌘ Command と スペース を同時に押す
  2. 画面上部に検索窓が出るので「ターミナル」と入力
  3. 「ターミナル」が表示されたら Enter を押す
  4. 黒または白いウィンドウが開いたらOK

📍 Windowsでターミナルを開く方法

  1. 画面左下のスタートボタンを右クリック
  2. 「Windows PowerShell」または「ターミナル」をクリック
英語の文字が流れても大丈夫： コマンドを実行すると英語がたくさん流れますが、エラーでなければ問題ありません。最後の行が $ や > で終わっていれば「待機中（次のコマンドを受け付けています）」のサインです。

セットアップ手順

1

ツールの土台をインストールする

これ以降のツールをスムーズに入れるための「基盤」です。最初に一度だけ行います。

ターミナルを開いて、以下をコピー&貼り付けして Enter を押してください。

Mac / Linux の場合
curl -LsSf https://astral.sh/uv/install.sh | sh
Mac で Homebrew を使っている場合はこちらでも OK
brew install uv
完了の確認： ターミナルを一度閉じて、もう一度開いてください。 その後 uv --version と入力して Enter。数字が表示されればOKです。
2

Browser Use をインストールする

Claude がブラウザを操作するための本体です。

uv tool install "browser-use[cli]"
完了の確認： browser-use --version を実行してバージョンが出ればOK。
3

自動操作用のブラウザをダウンロードする

Claude が裏で使う専用のブラウザをダウンロードします。あなたが普段使うChromeとは別物なので、普段の設定には影響しません。

playwright install chromium
「playwright が見つかりません」と出た場合：
まず uv tool install playwright を実行してから、もう一度上のコマンドを試してください。
4

Claude を動かすための「鍵」を設定する

Claude を使うには、自分専用の「APIキー」が必要です。これは Claude に「あなたのアカウントで動いてください」と伝えるための合言葉です。

① まず APIキー を取得する

  1. console.anthropic.com を開く（Googleアカウントでログインできます）
  2. 左のメニューから「API Keys」をクリック
  3. 「Create Key」ボタンを押す
  4. sk-ant- で始まる長い文字列が表示されるので、必ずコピーしてメモ帳などに保存（このページを閉じると二度と見られません）

② パソコンにキーを記憶させる

以下のコマンドの sk-ant-ここにキーを貼る 部分を、取得したキーに置き換えてから実行してください。

Mac / Linux の場合（2行を順番に実行）
echo 'export ANTHROPIC_API_KEY="sk-ant-ここにキーを貼る"' >> ~/.zshrc
source ~/.zshrc
Windows の場合
$env:ANTHROPIC_API_KEY="sk-ant-ここにキーを貼る"
ポイント： sk-ant-ここにキーを貼る の部分だけを書き換えてください。前後の "（ダブルクォーテーション）は消さないようにしてください。
5

試しに動かしてみる

ここまでの設定が正しくできているか確かめます。

browser-use "Googleを開いて、検索窓に「Browser Use」と入力して"
成功すると： 自動でブラウザが開いて、Googleで検索が実行されます。これが確認できればセットアップ完了です！
エラーが出た場合： エラーメッセージをそのままコピーして、Claude（またはChatGPT）に「このエラーどうすればいい？」と貼り付けて聞くのが一番早い解決方法です。

フォーム営業で実際に使う

セットアップが終わったら、こんなふうに使えます。 まずは 1社だけで試してみることをおすすめします。動作を確認してから、複数社に広げてください。

1社で試す場合
browser-use "https://example.com/contact のお問い合わせフォームに
会社名『B4B』、担当者名『山田』、
メッセージ『サービスについて詳しくお聞きしたいです』
と入力して、送信ボタンを押してください"
CSVリストで複数社に一括送信する場合
browser-use "companies.csv を読み込んで、
各企業のお問い合わせフォームに
会社名『B4B』、担当者名『山田』、
メッセージ『〇〇についてご相談したいです』
と入力して送信してください。
送信できた企業名を result.csv に記録してください"
companies.csv について： 企業のURLを縦に並べたExcelファイル（CSV形式）を用意してください。作り方が分からない場合は、ChatGPTに「企業URLをまとめたCSVファイルの作り方を教えて」と聞くとすぐ教えてくれます。

よくある質問

Q. パソコンの知識がなくても大丈夫？

このページの手順を順番に進めれば問題ありません。分からない言葉が出てきたら、そのままChatGPTや Claude に「〇〇ってどういう意味？」と聞いてください。

Q. エラーが出て止まった。どうすればいい？

ターミナルに表示されているエラーメッセージをすべてコピーして、Claude または ChatGPT に「このエラーが出ました。どうすれば解決できますか？」と貼り付けて聞いてください。たいていすぐ解決します。

Q. 費用はどのくらいかかる？

フォーム送信の件数と内容によりますが、シンプルな内容で100件あたり数十〜数百円程度が目安です。最初は少ない件数で試して、実際のコストを確認することをおすすめします。

Q. 普段使っているChromeのパスワードやログイン情報は影響を受ける？

影響しません。Browser Use は専用の別ブラウザを使うので、普段のChromeの設定やデータには一切触れません。

Q. Macじゃないと使えない？

Windows でも使えます。手順の一部（ターミナルの開き方やコマンドの書き方）が異なりますが、このページに両方の方法を記載しています。
