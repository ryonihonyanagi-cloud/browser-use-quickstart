Browser Use とは？

AIがChromeを直接操作できるようにするオープンソースのツール。「このフォームを入力して送信して」と指示するだけで、Claude Code が実際に画面を見ながら操作してくれる。

インストール手順

1

uv をインストールする

Python製ツールの高速インストーラー。まずこれを入れる。

# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Homebrew が好きな人はこちら
brew install uv

インストール後、ターミナルを再起動するか source ~/.zshrc を実行する。

2

Browser Use CLI をインストールする

uv tool install "browser-use[cli]"
確認： browser-use --version でバージョンが表示されればOK。
3

Playwright のブラウザをセットアップ

Browser Use が使うChromeの自動化エンジンを入れる。

playwright install chromium
4

APIキーを設定する

Claude Code（Anthropic）のAPIキーを環境変数に設定する。

# ~/.zshrc または ~/.bashrc に追記
export ANTHROPIC_API_KEY="sk-ant-..."

# 反映
source ~/.zshrc
APIキーの取得： console.anthropic.com → API Keys から発行する。
5

動作確認

試しに動かしてみる。

browser-use "Google を開いて、検索窓に『Browser Use』と入力して"

Chromeが自動で動き出せばインストール完了。

トークン節約のコツ（Browser Use 編）

既存のChromeに接続する

毎回新しいブラウザを起動するとコストがかかる。--cdp-url で既存のChromeに接続すると起動コスト不要で高速。

browser-use --cdp-url http://localhost:9222 "..."

タスクを細かく分割する

「全部やって」より「フォームを見つける」→「入力する」→「送信する」と分けた方がトークンが少なくて済む。失敗したときの修正も楽になる。
