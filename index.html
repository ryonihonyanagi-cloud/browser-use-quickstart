B4B 自動化セットアップガイド

フォーム営業を
Claude に任せる

企業のURLリストを渡すだけで、フォームへの入力・送信・記録まで自動でやってくれます。このページを読み終わる頃には、エンジニアが使う技術用語もいくつか分かるようになっています。

⏱ 所要時間：30〜60分（初回のみ） 🧠 用語解説つき

何ができるようになるの？

「Browser Use（ブラウザ・ユーズ）」というツールを使うと、Claude Code がパソコンのブラウザを代わりに操作してくれます。 あなたが企業のURLリストを渡して「このフォームに送って」と指示するだけ。フォームを探して、内容を入力して、送信ボタンを押すまで、すべて自動でやってくれます。

→
🔍 フォーム検出
→
✍️ 入力・送信
→
📊 結果を記録
💡 ちょっと豆知識
Browser Use は2025年3月に大幅アップデートされ、速度が従来の2倍・コストが半額になりました。 「CDP（シーディーピー）」という技術を使っていて、すでに開いているChromeにそのまま接続できるのが特徴です。 CDPとは「Chrome DevTools Protocol」の略で、ブラウザを外部のプログラムから操作するための仕組みです。 エンジニアが日常的に使っている技術です。

「ターミナル」を開いてみよう

これ以降の作業はすべて ターミナル という場所で行います。

📖 用語解説
ターミナル
パソコンに「文字で命令を出す」ための画面。昔のパソコンはすべてこの方法で操作していました。黒い画面に文字が流れるあれです。GUIが登場する前の操作方法ですが、エンジニアは今でもこれを使います。
コマンド
ターミナルに打ち込む「命令文」のこと。英語で書かれていますが、基本はコピペするだけなので読み解く必要はありません。
💡
この先に出てくる黒い画面のコードは、すべてコピーして貼り付けるだけでOKです。 自分で打ち込む必要はありません。貼り付けたら Enter を押すだけです。

📍 Macでターミナルを開く方法

  1. ⌘ Command + スペース を同時に押す
  2. 「ターミナル」と入力 → Enter
  3. 黒か白のウィンドウが開いたらOK

📍 Windowsでターミナルを開く方法

  1. スタートボタンを右クリック
  2. 「Windows PowerShell」または「ターミナル」を選択
英語が流れても焦らない： コマンドを実行すると英語がたくさん流れますが、正常です。最後の行が $ や > で終わっていれば「次のコマンドを待っている状態」です。

セットアップ（5ステップ）

1

uv をインストールする

📖 用語解説
uv
Python製のツールを素早くインストール・管理するためのツール。「apt」や「brew」と同じ「パッケージマネージャー」と呼ばれるカテゴリのツールです。まずこれを入れると、この後の作業がスムーズになります。
パッケージマネージャー
ソフトウェアの「インストールと管理」を自動でやってくれる仕組み。iPhoneの App Store のような存在です。
Mac / Linux の場合
curl -LsSf https://astral.sh/uv/install.sh | sh
Homebrew を使っている場合はこちらでも OK
brew install uv
完了の確認： ターミナルを一度閉じて開き直す → uv --version を入力 → バージョン番号が出ればOK。
✅ ステップ1完了！パッケージマネージャーを手に入れた
2

Browser Use をインストールする

📖 用語解説
CLI（シーエルアイ）
「Command Line Interface」の略。マウスを使わず、文字（コマンド）で操作するタイプのソフトウェアのこと。今回インストールするのは Browser Use の CLI 版です。
オープンソース
設計図（ソースコード）が公開されているソフトウェア。誰でも無料で使えて、世界中の開発者が改善に参加できます。Browser Use もオープンソースです。
uv tool install "browser-use[cli]"
完了の確認： browser-use --version でバージョンが出ればOK。
✅ ステップ2完了！AI自動化ツールの本体が入った
3

自動操作用のブラウザエンジンをダウンロードする

📖 用語解説
Playwright（プレイライト）
Microsoftが開発した「ブラウザを自動操作する」ためのツール。Browser Use は裏側でこれを使っています。あなたが普段使うChromeとは別の、AI専用のブラウザエンジンをここでダウンロードします。
Chromium（クロミウム）
Google Chromeの設計図をオープンソース化したもの。Chromeの「エンジン部分」だけを取り出したイメージです。今回はこれをAI操作専用として使います。
playwright install chromium
「playwright が見つかりません」と出た場合：
uv tool install playwright を先に実行してから、もう一度上のコマンドを試してください。
✅ ステップ3完了！AI専用ブラウザの準備ができた
4

APIキーを設定する

📖 用語解説
API（エーピーアイ）
「Application Programming Interface」の略。あるソフトウェアの機能を、別のソフトウェアから呼び出すための「窓口」です。Claude の API を使うと、自分のプログラムから Claude を動かせます。
APIキー
APIを使うための「鍵」となる文字列。あなた専用のパスワードのようなもので、「このキーを持つ人だけ Claude を使える」という認証に使います。
環境変数
パソコン全体で共有する「設定値の保管場所」。ここにAPIキーを登録しておくと、あらゆるツールが自動でキーを読み取れます。毎回手入力しなくて済む仕組みです。

① APIキーを取得する

  1. console.anthropic.com を開く（Googleアカウントでログイン可）
  2. 左メニュー「API Keys」→「Create Key」
  3. sk-ant- で始まる文字列をコピーしてメモ帳に保存
注意： このページを閉じると二度と表示されません。必ずその場で保存してください。

② 環境変数に登録する

以下の sk-ant-ここにキーを貼る の部分を実際のキーに書き換えてから実行してください。

Mac / Linux（2行を順番に実行）
echo 'export ANTHROPIC_API_KEY="sk-ant-ここにキーを貼る"' >> ~/.zshrc
source ~/.zshrc
Windows PowerShell
$env:ANTHROPIC_API_KEY="sk-ant-ここにキーを貼る"
💡 なぜこの作業が必要なの？
Browser Use が Claude を呼び出すとき、「どのアカウントで使うか」を確認するためにAPIキーを参照します。環境変数に登録しておけば、毎回入力しなくてもOK。エンジニアが日常的にやっている作業です。
✅ ステップ4完了！Claude と通信できるようになった
5

動作確認

ここまでの設定が正しくできているか確かめます。

browser-use "Googleを開いて、検索窓に「Browser Use」と入力して"
成功すると： ブラウザが自動で起動して、Googleで検索が実行されます。これでセットアップ完了です！
エラーが出た場合： エラーメッセージをそのままコピーして、Claude か ChatGPT に「このエラーどうすればいい？」と貼り付けて聞いてください。
🎉 セットアップ完了！フォーム営業を自動化する準備が整いました

フォーム営業で実際に使う

まずは1社だけで動作確認してから、複数社に広げることをおすすめします。

1社で試す場合
browser-use "https://example.com/contact のお問い合わせフォームに
会社名『B4B』、担当者名『山田』、
メッセージ『サービスについて詳しくお聞きしたいです』
と入力して、送信ボタンを押してください"
CSVリストで複数社に一括送信する場合
browser-use "companies.csv を読み込んで、
各企業のお問い合わせフォームに
会社名『B4B』、担当者名『山田』、
メッセージ『〇〇についてご相談したいです』
と入力して送信してください。
送信できた企業名を result.csv に記録してください"
📖 用語解説
CSV（シーエスブイ）
「Comma-Separated Values」の略。Excelで「CSV形式で保存」するとできるファイル。企業URLのリストをExcelで作り、CSVで保存してフォルダに置いておけば、Claude が自動で読み込んでくれます。
💡 コストを抑えるコツ

Claude Code でのコストは「トークン数（やり取りの量）」に比例します。フォーム営業なら「フォームを探す」→「入力する」→「送信する」とタスクを小分けにすると、失敗したときの修正も楽で、トークンも節約できます。

よくある質問

Q. エラーが出て進めない

ターミナルに表示されているエラーメッセージ（英語でOK）をすべてコピーして、Claude か ChatGPT に「このエラーどうすればいい？」と貼り付けてください。たいていすぐ解決します。

Q. 普段使いのChromeのデータに影響する？

影響しません。Browser Use は「Chromium」という専用の別ブラウザを使います。普段のChromeのパスワードやブックマークには一切触れません。

Q. 費用はどのくらいかかる？

シンプルなフォーム100件あたり、数十〜数百円が目安です。最初は少ない件数で試してコストを確認するのがおすすめです。

Q. このページを読んだだけでエンジニアになれる？

なれませんが、「API」「CLI」「環境変数」「CDP」「オープンソース」といった言葉の意味は分かるようになります。エンジニアと話すときに「何を言っているか」が分かるようになるのが第一歩です。
