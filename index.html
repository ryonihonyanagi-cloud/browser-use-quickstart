Browser Use を使えば、Claude Code がブラウザを自動操作してフォーム送信までやってくれる。難しい設定は不要。このページの手順通りに進めればOK。

⚡ 速度2倍・コスト半額（2025年3月） 🖥️ Mac / Windows 対応

Browser Use って何？

AIがChromeを「目で見て・手で操作する」ように動かせるオープンソースのツールです。 Claude Code に「このフォームに入力して送信して」と日本語で指示するだけで、実際に画面を操作してくれます。

始める前に確認すること

✓

ターミナル（黒い画面）の開き方

これ以降の手順は、すべてターミナルに文字を入力して進めます。

📌 Macの場合

  1. キーボードで ⌘ Command + スペース を押す
  2. 「ターミナル」と入力して Enter
  3. 黒または白いウィンドウが開いたらOK

📌 Windowsの場合

  1. スタートメニューを右クリック
  2. 「Windows PowerShell」または「ターミナル」をクリック
コマンドの実行方法： この後に出てくるコードブロック（黒い背景の部分）をコピーして、ターミナルに貼り付けて Enter を押すだけです。
✓

Anthropic の API キーを用意する

Claude Code を動かすための「鍵」です。まだ持っていない場合は先に取得してください。

  1. console.anthropic.com にアクセス（Googleアカウントでログイン可）
  2. 左メニューの「API Keys」→「Create Key」
  3. sk-ant-... で始まる文字列をメモ帳などに控えておく
注意： APIキーは一度しか表示されません。必ずその場でコピーしてどこかに保存してください。

インストール手順（5ステップ）

1

uv をインストールする

Python 製ツールを簡単に管理できるインストーラーです。まずこれを入れます。

# Mac / Linux の場合
curl -LsSf https://astral.sh/uv/install.sh | sh
# Homebrew を使っている場合はこちらでも OK
brew install uv
完了の確認： コマンドが終わったら、一度ターミナルを閉じて再度開いてください。その後 uv --version と入力して、バージョンが表示されればOKです。
2

Browser Use をインストールする

uv tool install "browser-use[cli]"
完了の確認： browser-use --version と入力してバージョンが表示されればOK。
3

ブラウザ自動化エンジンをセットアップする

Browser Use が使う専用のChromeエンジンをダウンロードします。少し時間がかかります。

playwright install chromium
「playwright が見つかりません」と出た場合：
uv tool install playwright を先に実行してから、再度上のコマンドを試してください。
4

APIキーを設定する

さきほど取得した sk-ant-... のキーを設定します。

# Mac / Linux の場合（以下を1行ずつ実行）
echo 'export ANTHROPIC_API_KEY="sk-ant-ここにキーを貼る"' >> ~/.zshrc
source ~/.zshrc
# Windows PowerShell の場合
$env:ANTHROPIC_API_KEY="sk-ant-ここにキーを貼る"
注意： sk-ant-ここにキーを貼る の部分を、実際のAPIキーに置き換えてください。クォート（ " ）は消さないこと。
5

動作確認

試しに動かしてみましょう。Chromeが自動で操作されれば成功です。

browser-use "Googleを開いて、検索窓に『Browser Use』と入力して"
成功すると： 自動でChromeが起動して、Googleで検索を実行します。これができればインストール完了です。

フォーム営業への応用

自動化できる一連の流れ

📋 企業URLリスト
→
🔍 フォーム検出
→
✍️ 自動入力・送信
→
📊 結果を記録

企業URLを並べたCSVファイルを用意して、以下のように Claude Code に指示するだけで動きます。

browser-use "companies.csv を読み込んで、各企業のお問い合わせフォームに
会社名『B4B』、担当者名『山田』、メッセージ『〇〇についてご相談したい』
と入力して送信してください。送信できた企業を result.csv に記録してください。"
まず小さく始めるのがおすすめ： 最初は1社だけのURLで試してみてください。動作を確認してから、複数社に広げると安心です。

よくある質問

Q. Pythonは必要ですか？

uv が自動で管理するので、Pythonを別途インストールする必要はありません。

Q. エラーが出て進めない場合は？

エラーメッセージをそのままコピーして Claude Code（または ChatGPT）に貼り付けて「これどうすればいい？」と聞くのが一番早いです。

Q. 既存のChromeをそのまま使えますか？

はい。--cdp-url http://localhost:9222 オプションで、ログイン済みのChromeに直接接続できます。サイトへのログインも引き継がれます。

Q. コストはどのくらいかかりますか？

タスクの複雑さとフォーム数によります。目安として、シンプルなフォーム100件あたり数十円〜数百円程度です。/compact コマンドでセッションを圧縮するとコストを抑えられます。
